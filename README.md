# netflix_data_analysis

Proyecto centrado en un análisis exploratorio de datos (EDA) y visualización sobre una base de datos de Netflix.

## Objetivo
Aplicar técnicas de:
- análisis exploratorio de datos,
- tratamiento/limpieza de datos,
- análisis de datos,
- visualización de datos.

## Archivo principal
- `netflix_data_analysis.ipynb`

## Requisitos sugeridos
Para ejecutar el notebook se recomienda contar con:
- Python 3
- pandas
- matplotlib
- seaborn

Además, coloca el dataset `netflix_titles.csv` en alguno de estos paths:
- `netflix_titles.csv`
- `data/netflix_titles.csv`
- `dataset/netflix_titles.csv`
